To-do List Improved

Code Firebase Version without login

The Why:

I built this to gain the experience of working within an existing code structure to implement functionality, code and accessibility improvements. The project started with the code here: Original (click run at the top) and progressed from there. I implemented local storage, simplified adding, deleting and changing a todo, provided styling, and worked to improve accessibility(such as color accessibility, adding labels to check boxes and forms).

Tech used:

HTML/CSS, JavaScript, local storage, Firebase, Firebase Authentication, Firebase Database

Challenges and things to improve:

After building a version using local storage I wanted to over complicate things in efforts to learn more. In this process, another version of this project was born using the Firebase platform and the authentication and real time database features. I would like to further improve the accessibility of this To-do app.
